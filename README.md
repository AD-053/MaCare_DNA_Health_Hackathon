# 🏥 MaCare - Digital Maternal & Child Healthcare Platform

A comprehensive digital health solution designed to connect mothers, doctors, and midwives in Bangladesh, ensuring every mother and child receives quality healthcare.

> **Developed during DNA Hack For Health 2026** | MAG Osmani Medical College, Sylhet | January 16, 2026

---

## 📋 Problem Statement

Bangladesh faces critical challenges in maternal and child healthcare:

- 🚨 **High Maternal Mortality**: 156 deaths per 100,000 live births (mostly preventable)
- 📉 **Limited Rural Access**: 65% of rural population struggle to access healthcare
- 🤰 **Missed Check-ups**: Many pregnant women miss critical antenatal care
- 👶 **Child Nutrition Crisis**: 24% of children under five are stunted
- 📱 **Connectivity Gap**: Limited health information reaches remote areas

**Result**: Thousands of preventable deaths every year.

---

## 💡 Our Solution

**MaCare** is a mobile-first digital platform that bridges the gap between mothers, healthcare providers, and medical professionals.

### Key Features

✅ **Pregnancy Tracking** - Monitor crucial pregnancy milestones and health metrics  
✅ **Doctor Consultations** - Connect with doctors remotely without long-distance travel  
✅ **Vaccination Management** - Get reminders and track child vaccinations  
✅ **Emergency Alerts** - Real-time notifications for critical health situations  
✅ **Health Education** - Free educational content in Bengali  
✅ **Role-Based Dashboards** - Separate interfaces for mothers, doctors, and midwives  
✅ **Secure Data Management** - HTTPS encryption and secure health records  

---

## 🎯 Three-User Ecosystem

### For Mothers (মা)
- Track pregnancy progress and health metrics
- Book and manage doctor appointments
- Receive vaccination reminders
- Access free health education content
- Monitor child growth and nutrition
- Get emergency support when needed

### For Doctors (ডাক্তার)
- Manage patient database and medical history
- Provide remote consultations
- Monitor high-risk pregnancies
- Issue health advice and prescriptions
- Track patient health trends

### For Midwives (মিডওয়াইফ)
- Conduct weekly check-ups
- Monitor vital signs and health metrics
- Manage up to 20 mothers per midwife
- Coordinate community-level care
- Report to doctors for critical cases

## 🛠️ Tech Stack

### Frontend
- **React** (v19.2.0) - UI Framework
- **Vite** (v7.2.4) - Build Tool & Development Server
- **Tailwind CSS** (v3.4.19) - Styling & Design System
- **Lucide React** - Icon Library

### Backend
- **Node.js** - JavaScript Runtime
- **Express.js** (v5.1.0) - Web Framework
- **MongoDB** (via Mongoose v8.18.3) - NoSQL Database
- **Socket.io** (v4.7.5) - Real-time Communication

### Authentication & Security
- **JWT** (jsonwebtoken v9.0.2) - Token-based Authentication
- **Bcrypt** (v6.0.0) - Password Hashing
- **CORS** - Cross-Origin Resource Sharing

### Additional Services
- **Cloudinary** (v2.7.0) - Image & File Storage
- **Multer** (v2.0.2) - File Upload Management
- **Nodemailer** (v7.0.10) - Email Service
- **Socket.io** - Real-time Notifications
- **Mongoose Aggregate Paginate** - Database Pagination

### Development Tools
- **Nodemon** - Auto-restart during development
- **Prettier** - Code Formatting
- **ESLint** - Code Linting
- **PostCSS** & **Autoprefixer** - CSS Processing
